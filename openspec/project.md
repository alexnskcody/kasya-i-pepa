# Проект: Кася и Пепа

## Контекст

Детская браузерная игра-игрушка: два питомца (бурма Кася и цвергшнауцер
Пепа), срисованные с настоящих фото. Целевая платформа — телефон
(iOS Safari в первую очередь), добавление на главный экран как PWA-лайт.
Live: https://alexnskcody.github.io/kasya-i-pepa/

## Ограничения

- Чистые HTML+CSS+JS, ноль зависимостей, без сборки. Никаких package.json,
  бандлеров, CDN — игра работает офлайн после первой загрузки.
- Хостинг — статика на GitHub Pages, `.nojekyll` обязателен.
- Автотестов нет; верификация — запуском (скилл run-game), iOS-специфику
  проверять на реальном устройстве.
- Аудио обязано переживать iOS Lockdown Mode: фолбэк — настоящие
  HTTP-файлы AAC (.m4a) из `audio/`, генерируемые `tools/gen_audio.py`.

## Конвенции

- Все артефакты — коммиты, спеки, изменения, доки — НА РУССКОМ.
- Спеки возможностей лежат в `openspec/specs/`, активные изменения —
  в `openspec/changes/<id>/` (proposal.md, tasks.md, design.md при
  необходимости). Обычные файлы — контракт; CLI openspec не установлен.
- Инварианты проекта — в CLAUDE.md, раздел «Инварианты».
