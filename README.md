# Кася и Пепа 🐾

Детская браузерная игра-игрушка с двумя питомцами, срисованными с настоящих фото:
шоколадной бурмой **Касей** и чёрным цвергшнауцером **Пепой**.

Без установки, без интернета, без рекламы. Один раз открыл — играй.

## Как запустить

**На компьютере:** просто открой `index.html` двойным кликом (работает прямо с диска).

**На телефоне (в той же Wi-Fi сети):**
```bash
cd pets-game
python3 -m http.server 8317
```
Узнай IP компьютера (`ipconfig getifaddr en0` на Mac) и открой на телефоне
`http://<IP-компьютера>:8317`. Добавь на главный экран — будет как приложение.

**Через GitHub Pages:** залей содержимое этой папки в репозиторий (`index.html` в корне),
в настройках репозитория включи Pages → Deploy from branch → `main` / root.
Игра появится на `https://<логин>.github.io/<репозиторий>/`.

## Что умеют питомцы

- **Потыкай** — мяукают, гавкают, подпрыгивают. Спамить тоже весело: у Каси лопается терпение, Пепа крутится волчком.
- **Погладь** (проведи пальцем) — Кася жмурится и мурчит, Пепа блаженно вываливает язык. Сердечки!
- **Сами по себе:** Кася валяется пузом кверху, вылизывается, спит калачиком.
  Пепа сидит и смотрит на тебя просящим взглядом, служит, нюхает пол, спит плашмя.
- **Фирменная сценка:** Кася подкрадывается и **БАП** лапой по носу — Пепа гоняется,
  ловит, комичное облако потасовки, Кася вырывается и удирает на когтеточку. Потом всё сначала 😄
- **Мячик** — толкни: Пепа приносит обратно, Кася подкарауливает и цапает.
- **Вкусняшка** — ближайший питомец прибежит и схрумкает.
- **Дружбометр** — наполни сердечко играми и ласками: будет салют и нос-к-носу!

## Режимы

| Кнопка | Что происходит |
|---|---|
| 💞 Вместе | Питомцы живут сами, ты тыкаешь, гладишь, кормишь |
| 🐱 Кася | Тапай по полу — бегай. Подкрадись к Пепе и жми «БАП!», потом удирай на когтеточку! |
| 🐶 Пепа | Кася сама дразнится — лови её! Догнал — тапай по облаку потасовки. Есть «Служить» 🥺 |

## Технически

Чистые HTML+CSS+JS (SVG-персонажи, Web Audio синтез звуков), ноль зависимостей,
~90 КБ. Пойнтер-ивенты: работает и мышью, и пальцем. Портрет и ландшафт.

Файлы: `index.html` (сцена), `css/style.css` (анимации), `js/art-*.js` (позы персонажей),
`js/pet.js` (движение/взгляд), `js/director.js` (поведение и сценки), `js/audio.js`, `js/fx.js`, `js/main.js`.
Дизайн-док: `docs/design.md`.
